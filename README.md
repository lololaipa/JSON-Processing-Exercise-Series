# Як запускати рішення

1. Відкрити проєкт у VS Code
2. Відкрити термінал (Ctrl + `)
3. Перейти в папку з файлами (якщо потрібно)
4. Запустити потрібний файл командою:
node <назва_файлу>.js 3000

Наприклад:
node json_echo.js 3000

Як перевірити (Verify)
Для перевірки вправ використовується команда:
eu-node-basics verify <номер_вправи> <назва_файлу>
